### Quanty
Everything in the /quanty directory is legacy code.
To run quanty:
1. `cd quanty`
2. `bundle install`
3. The first time: `QWERTY_RUN_MIGRATIONS=true bundle exec ruby ./quanty.rb`
5. Then every other time: `bundle exec ruby ./quanty.rb`
